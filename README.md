# Muhsalmon Personal Website v2

A personal portfolio site built with React, TypeScript, Tailwind CSS, and GSAP. Dark-only editorial design with smooth animations, responsive layout, and accessible UI.

## Documentation

All work in this repo must follow:

- [`AGENTS.md`](./AGENTS.md) — agent & code conventions
- [`design.md`](./DESIGN.md) — design system (editorial × Swiss × dark)
- [`CONTENT.md`](./CONTENT.md) — canonical copy, facts, and section content
- [`README.md`](./README.md) — this file

## Summary

Single-page portfolio with:

- **Hero** — Intro, CTAs, social links, scroll hint
- **About** — Bio and skills with animated text
- **Work** — Project cards with tech tags and demo/Repo links
- **Contact** — Form and contact details
- **Navigation & Footer** — Nav with theme toggle and site footer

Tech: React 19, Vite, Tailwind, Radix UI, Lucide icons, next-themes, GSAP. Mobile-first, performant, and WCAG-friendly.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Production build: `npm run build`.
